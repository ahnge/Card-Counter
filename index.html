Card Counting made easy

Card counting is a blackjack strategy used to determine whether the player or the dealer has an advantage on the next hand.My program uses Hi-Lo Strategy which is as follows.

Cards Values
2,3,4,5,6 +1
7,8,9 +0
10,J,Q,K -1

If you are using with computer, you can use keyboard for Ace(card) as 1, Two(card) as 2, Three(card) as 3, Jack(card) as j, Queen(card) as q, King(card) as k,etc...
